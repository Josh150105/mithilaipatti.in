# Mithilaipatti Village Website

A static website for the village of **Mithilaipatti** in Sivagangai district, Tamil Nadu.

## Project purpose

This repository contains an informational brochure-style website for the village, highlighting:

- history and culture
- places of worship (temple and church)
- tourism and local attractions
- festivals and events
- travel information and bus timings
- contact details

The site is bilingual with Tamil and English content.

## Root files

- `index.html`
  - Main landing page for the site
  - Contains inline CSS and page markup
  - Includes SEO metadata, Open Graph/Twitter cards, and structured schema data
  - Uses a language toggle, responsive navigation, and scroll-based sections

## Upgrade version

The `upgrade/` directory contains a newer, modular version of the same website:

- `upgrade/index .html`
  - Note: the filename includes a space and may need renaming to `index.html`
- `upgrade/style.css`
  - External stylesheet for layout and responsive styles
- `upgrade/script.js`
  - JavaScript for interactive features
  - Includes custom cursor, navigation menu, language toggle, hero particles, email form support, bus timing highlights, and analytics widgets

## Assets

The repository also includes image assets used by the site, including photos for:

- temple
- church
- local village scenes
- tourism highlights

## Features

- bilingual Tamil / English UI
- responsive layout for mobile, tablet, and desktop
- custom cursor and visual animation effects
- smooth anchor scrolling and sticky header navigation
- Google Analytics integration
- contact form and EmailJS support in upgraded version
- bus schedule and travel directions section

## How to view

Open either `index.html` or `upgrade/index .html` in a browser.

If using the upgraded version, consider renaming `upgrade/index .html` to `upgrade/index.html` for easier access.

## Notes

- This is a purely static website; there is no backend code in the repository.
- The `upgrade/` folder appears to be a cleaner refactor of the root site.
